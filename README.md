# sistema-de-gestion-de-conductores-go
Este proyecto es una simulación de un sistema de gestión de conductores y pedidos, similar a Uber, implementado en Go. Utiliza concurrencia a través de goroutines y canales para manejar múltiples conductores y pedidos de pasajeros de forma simultánea. El sistema asigna pedidos al conductor más cercano y simula el tiempo de viaje.
